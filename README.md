Diese Webseite wurde im Rahmen einer Fallstudie entwickelt und dient die transparente Darstellung von C02 Emissionsdaten von Ländern und Unternehmen Weltweit.
Projektziele: Entwicklung einer responsiven und benutzerfreundlichen Website, Implementierung moderner Webanwendungen, Sicherstellung von Accessiblity, Filter- und Sortierfunktionen für Daten, Mobil First Responsive Design
Features: Interaktive Datentabelle mit 30 fiktiven Datensätze, Live Suche, Responsives Desgin für alle Endgeräte, LTR/RTL Unterstützung.
Projrktstruktur: eine index.html Datei, eine style.css Datei, eine data.js Datei für die Datent, eine util.js Datei für die Hilfsfunktionen und eine app.js Datei für die Hauptanwendungslogik.
Verwendete Technologien: HTML5 für die semantische Struktur, CSS3 für das Styling und Grid, JavaScript für die Interaktivität, Git für die Versionsverwaltung und GitHub Pages für das Hosting.
